constexpr uint32_t c74::min::hash ( const char *const  str,
const uint32_t  seed = 0xAED123FD 
)
inlineconstexprnoexcept

A hash function using the Murmur3 algorithm ( https://en.wikipedia.org/wiki/MurmurHash ). This hash function is capable of being executed at compile time, meaning that the compiled binary will have a constant int value and no actually need to execute any code at runtime.

Parameters
strA c-string to be hashed into an int.
seedAn optional seed value. For most uses you should not override the default.
Returns
An int (specifically a uint32_t) representing the hash of the string input.

Detailed Description

template<timer_options options>
class c74::min::timer< options >

The timer class allows you to schedule a function to be called in the future using Max's scheduler. Note: the name timer was chosen instead of clock because of the use of the type is clock is ambiguous on the Mac OS when not explicitly specifying the c74::min namespace.

Template Parameters
optionsOptional argument to alter the delivery from the scheduler thread to the main thread.

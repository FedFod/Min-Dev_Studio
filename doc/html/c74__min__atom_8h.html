◆ atoms

using c74::min::atoms = typedef std::vector<atom>

The atoms container is the standard means by which zero or more values are passed. It is implemented as a std::vector of the atom type, and thus atoms contained in an atoms container are 'owned' copies... not simply a reference to some externally owned atoms.

◆ order

using c74::min::order = typedef int

Defines attribute ordering in the inspector. A value zero means there is no special order and Max will take care of the ordering automatically.

◆ range

using c74::min::range = typedef atoms

The range provides a definition of acceptable or 'normal' values for an attribute. Unless specified as a template-parameter to the attribute, the range is only a suggestion to the user. Ranges for numeric types will be two numbers (a low bound and a high bound). For symbols or enums the range will provide the available options.

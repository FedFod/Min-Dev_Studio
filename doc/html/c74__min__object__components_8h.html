std::string c74::min::deduce_maxclassname ( const char *  maxname)
inline

Deduce the intended name of a Max object from the name of the c++ sourcecode file. This is used internally via the MIN_EXTERNAL macro when the max::t_class is created. Thus the maxname parameter may come in as an entire path because of use of the FILE macro that is invoked.

Parameters
maxnameThe filename, possible the fullpath, of the c++ source file.
Returns
A string with the deduced object name as it will be exposed to the Max environment.

Parameters
fileusage_handleThe raw arguments passed to the "fileusage" message by Max.
package_nameThe name of the package to be added to the standalone.
names_of_folders_to_includeOptional. The names of the folders in the package to add to the standalone. If none are provided then the entire package will be added.

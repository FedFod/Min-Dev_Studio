Detailed Description

template<thread_check check, thread_action action>
class c74::min::outlet< check, action >

An outlet for sending output from a Min object. Outlets are specialized with two optional template parameters. In most cases you will use the defaults. For example, outlet<>.

Template Parameters
checkDefine which threads are valid sources for calls to the outlet.
actionDefine what action is to be taken if the thread check fails.

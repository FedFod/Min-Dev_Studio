Detailed Description

template<bool audio_thread_access = true>
class c74::min::buffer_lock< audio_thread_access >

A lock guard and accessor for buffer~ access.

Template Parameters
audio_thread_accessMake this true if you will access the buffer~ from the audio thread. Otherwise make this false for access on other threads. The default is to access on the audio thread.
